CTRL+ALT+T - команда открытия терминала 
~ - символ домашней дериктории (папки)
/ - символ корневой директории (папки)
.. - обозначает предыдущую дерикторию
. - обозначает текущую дерикторию
CTRL+SHIFT+C - копировать в терминале
CTRL+SHIFT+V - вставить
CTRL+L - clear
1. ls (list) -> Выводит список всех дерикторий и файлов в текущей дериктории (рабоча дериктория)
ls -a ->  выводит список как и скрытых, так и обычных файлов и директорий
ls -l -> выводит список директорий и файлов с подробной информацией

2. cd (change directory) -> меняет текущую дерикторию на указанную
cd<название директории>
3. mkdir (make directory) -> команда которя создаёт указанную директорию
mkdir <директория>
4. touch -> создаёт указанный файл
touch <name_of_file_>. <extension (расширение (txt,docs))> 
5. pwd (print working directory) -> выводит путь до текущей директории
6. rmdir remove directory) -> удаляет указанную директорию (не может удалить не пустую директорию)
7. rm -> удаление файла
rm <name_of_file>
rm -rf -> принудительное удаление файла или директории
8. mv (move) -> перемещает либо переименовывает файл или директорию в зависимости от указанного второго аргумента 
mv <файл_директория> <новое_имя -- директория/ > 
9. cat -> выводит содержимое указанного файла 
10. nano -> открывает графический интерфейс в терминале для редактирования файлов 
* если нет указанного файла, то создатся новый файл с таким именем
nano <name_of_file>
11. echo -> команда для быстрой записи данных в файле через терминал
echo текст для записи > название файла
12. cp (copy) - команда для kопирования файла в нужную директорию
cp <file_to_copy> <path>
